# 🛒 E-commerce Web Application

A full-stack e-commerce website built with:

- **Backend**: Spring Boot (Java 17), REST APIs, H2 in-memory database
- **Frontend**: Vue.js (separate folder)

---

## 🔧 Backend Features

- User authentication (login)
- Product management
- Cart system
- Contact form
- JWT-based security
- CORS configuration
